# Privacy Policy

**Last Updated:** December 20, 2024

## 1. Overview

SecureKeep ("the App") prioritizes user privacy above all else. This Privacy Policy explains how the App handles user information.

## 2. Data Collection

**SecureKeep does NOT collect any personal information.**

- ❌ No personal identification information collected
- ❌ No usage statistics collected
- ❌ No analytics data collected
- ❌ No network connection

## 3. Data Storage

All data is stored exclusively on the user's device:

- **Local Storage**: All notes are encrypted and stored in the device's local database (Hive)
- **Encryption**: Military-grade security using AES-256-GCM algorithm
- **No Cloud**: Data is never transmitted to external servers

## 4. Biometric Authentication

The App may use biometric authentication (fingerprint, Face ID):

- **Purpose**: App unlock only
- **Processing**: Biometric data is processed only in the device's secure area (Secure Enclave/TEE)
- **Storage**: The App does not store or access biometric information

## 5. Permissions

Permissions requested by the App:

- **Biometric Permission**: Unlock app with fingerprint/Face ID
- **Storage Permission**: Access local database (device internal only)

**Permissions NOT requested:**
- ❌ Internet (no network connection)
- ❌ Location
- ❌ Camera
- ❌ Contacts

## 6. Data Deletion

- **App Uninstall**: All data is permanently deleted
- **No Recovery**: Data cannot be recovered as there is no cloud backup
- **User Responsibility**: Users must separately backup important data

## 7. Third-Party Services

**SecureKeep does NOT use any third-party services:**

- ❌ No advertisements
- ❌ No analytics tools (Google Analytics, Firebase, etc.)
- ❌ No social media integration
- ❌ No cloud synchronization

## 8. Children's Privacy

The App does not intentionally collect information from children under 13. If used by children, parental supervision is recommended.

## 9. Security

- **Encryption**: AES-256-GCM
- **Key Management**: Uses device's secure keychain (iOS Keychain, Android KeyStore)
- **Screenshot Prevention**: Blocks screenshots and screen recording on Android
- **Background Security**: Background snapshot masking on iOS

## 10. Policy Changes

This Privacy Policy may change. Significant changes will be communicated to users through app updates.
