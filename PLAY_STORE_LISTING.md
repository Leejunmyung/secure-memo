## 앱 제목 (30자 제한)

## 짧은 설명 (80자 제한)
완전 오프라인 암호화 메모장. 민감한 정보를 안전하게 보관하세요. 네트워크 연결 없음.

## 전체 설명 (4000자 제한)

SecureKeep - 당신만의 디지털 금고

완전한 오프라인 보안 메모장으로 비밀번호, 계정 정보, 카드 정보 등 민감한 데이터를 안전하게 보관하세요.

🔒 완전한 프라이버시
• 100% 오프라인 동작 - 인터넷 권한 없음
• 기기를 떠나지 않는 데이터
• 클라우드 동기화 없음
• 데이터 수집 없음
• 광고 없음

🛡️ 군사급 보안
• AES-256-GCM 암호화
• 생체 인증 (지문/얼굴 인식)
• PIN 잠금
• 스크린샷 방지
• 백그라운드 스냅샷 마스킹
• 클립보드 자동 삭제

📝 다양한 메모 타입
• 일반 메모: 자유로운 텍스트 입력
• 계정 정보: 아이디/비밀번호 안전 보관
• 카드 정보: 신용카드 정보 암호화 저장

✨ 편리한 기능
• 즐겨찾기 메모
• 빠른 검색
• 메모 타입별 필터
• 다크 모드 지원 (곧 출시)
• 한글 UI 완벽 지원

⚠️ 주의사항
• 앱 삭제 시 데이터 영구 손실
• PIN 분실 시 복구 불가능
• 클라우드 백업 없음 (100% 로컬 전용)
• 사용자가 직접 백업 권장

💡 왜 SecureKeep인가?
• 검증 가능한 오프라인: AndroidManifest에서 인터넷 권한 없음 확인 가능
• 구독료 없음: 평생 무료
• 개인정보 보호 최우선: 데이터가 당신의 기기를 떠나지 않습니다
• 단순함: 복잡한 설정 없이 즉시 사용 가능

📱 호환성
• Android 8.0 이상
• 태블릿 지원

🌐 오픈소스
• GitHub에서 소스 코드 확인 가능 (투명성 보장)
